# 北海道・新産業共創担当 LP — v0.2

## コンセプト（2026-07-08 刷新）

> 「私たちは北海道を中心に、主に新産業領域（オープンイノベーション・半導体・データ利活用等）にてみなさまと共創し、地域の事業を創出するチームです」

- **目的**: **発信がメイン**（チームの活動方針・「この人たちは何ができるのか」を伝える）。リード獲得はあわよくば
- **対象**: 北海道・新産業共創担当（まちづくり事業部の一機能）
- **役割**: 8月の会社全体リニューアルとは別の、チーム独自のLP

## 設計上の制約（外せないルール）

| # | 制約 | 理由 |
|---|---|---|
| 1 | **「事業所」「拠点」「オフィス」「支店」「営業所」「部」「課」表記NG** | 法人税の事業所扱いを避けるため。「北海道」地名そのものはOK。「事業として北海道で取り組む」表現に統一 |
| 2 | **エリア区切りではなく事業単位で構成** | 上記制約と、今後の事業追加への柔軟性のため |
| 3 | **「新産業」キーワードを軸に置く** | オープンイノベーション・半導体・データ利活用等を束ねる上位概念。汎用研修との差別化軸 |
| 4 | **AKATSUKIは別ページで掲載**（トップには載せない） | v0.2方針。補助金名は「未踏的な」を付けず「地方の若手人材発掘育成支援事業費補助金」 |
| 5 | **IOWN・核ゴミ案件は現時点で非公開** | 進行案件のため、LP掲載は時期判断後 |
| 6 | **デザインテイストは会社トップページ改修案に揃える** | `design-ref/` 内のPDF参照。会社全体との一貫性 |
| 7 | **社内向け文言（DISCUSSマーカー等）は載せない** | 対外発信用ページのため（v0.1のたたき台マーカーは削除済み） |

## 構造（6セクション）

```
HERO（ヒーロー）
  └ キャッチコピー + チーム定義文

ABOUT（私たちについて）
  └ 担当の位置づけ + 新産業の定義 + キーワードタグ

BUSINESS（取り組み事業）  ← LPの主役
  └ 地域社会DX推進・自治体伴走支援（NEW） / 半導体産業共創 / データ利活用・地域経済 / 制度支援AI・自治体実装

PROJECT FOCUS（注目プロジェクト）
  └ 総務省「地域社会DX推進パッケージ事業（推進体制構築支援）」の詳細
     （支援地域・期間・体制・3つの支援内容・展望・リリースリンク）

SEMINAR（セミナー・イベント）
  └ 定期更新枠（現在は「順次ご案内」の空枠）

CONTACT（お問い合わせ）
  └ 問い合わせCTA（リンク先は未設定 → 要確認）
```
