## このページの位置づけ（前提共有）

- **対象**: 北海道・新産業共創担当（まちづくり事業部の一機能）
- **役割**: 8月の会社全体リニューアルとは別に、**チーム独自のLP**を持つ
- **目的**:
  - 北海道で取り組む「新産業領域」の事業を、自治体・地域企業・大学に伝える
  - 定期セミナーの集客導線
  - 採用・問い合わせの入口
- **公開時期**: 担当議論を経て、宮本がClaudeに実装指示 → 公開準備

## 設計上の制約（外せないルール）

| # | 制約 | 理由 |
|---|---|---|
| 1 | **「事業所」「拠点」「オフィス」「支店」「営業所」「部」「課」表記NG** | 法人税の事業所扱いを避けるため。「北海道」地名そのものはOK。「事業として北海道で取り組む」表現に統一 |
| 2 | **エリア区切りではなく事業単位で構成** | 上記制約と、今後の事業追加への柔軟性のため |
| 3 | **「新産業」キーワードを軸に置く** | 半導体・データ・AKATSUKI・IOWN等を束ねる上位概念。汎用研修との差別化軸 |
| 4 | **AKATSUKI補助金名は「未踏的な」を付けず**「地方の若手人材発掘育成支援事業費補助金」 | 正式名称統一ルール |
| 5 | **IOWN・核ゴミ案件は現時点で非公開** | 進行案件のため、LP掲載は時期判断後 |
| 6 | **デザインテイストは会社トップページ改修案に揃える** | `design-ref/` 内のPDF参照。会社全体との一貫性 |

## 議論してほしいポイント（7つ）

LP本文中の **オレンジ枠（💬 DISCUSS）** にも書いてありますが、まとめます。

### ① HEROキャッチコピー
- 現状: 「北海道から、新産業の共創を。」
- **問い**: このコピーで「ASHIOTOやリスキリングと違うチーム」と伝わるか？ もっと尖らせるべきか、別軸（人物／景色／数字）でいくべきか？

### ② ABOUTの「新産業」定義の書きぶり
- 現状: 「半導体・データ利活用・若手起業家育成・IOWN等、日本の未来に関わる新産業領域」
- **問い**: 外部読者（自治体・地域企業の担当者）にこの定義は腹落ちするか？ もっと具体例を増やすか、抽象度を上げるか？

### ③ 事業カードの **並び順**
- 現状: AKATSUKI → 半導体 → データ → 制度支援AI
- **問い**: チームとして「先頭に立てたい事業」「最後に置きたい事業」は？

### ④ 事業カードの **キャッチ＆説明文**
- それぞれ「ひと言キャッチ」と「3行説明」を作りました
- **問い**: ニュアンスが違うところはありますか？ チームの言葉で書き直してください

### ⑤ 掲載する事業の **本数と分類**
- 現状: 4事業
- **問い**: 増やすべきか（例: HVVA、北海道庁案件、ANIC等）／統合すべきか／「現状実施中」と「準備中」を分けるか？

### ⑥ セミナーセクションの **運用**
- 現状: 3カード固定
- **問い**: 誰がどう更新する？ 月1更新？ 過去アーカイブはどこに？ 申込導線（Googleフォーム／PeaTix／Slack）は？

### ⑦ 写真・ビジュアル素材
- 現状: 全てプレースホルダー（グラデーション色面）
- **問い**: 各事業で使う写真は何を出すべき？ チームメンバー写真は出す／出さない？ Rapidusや札幌の風景は使えるか？
